# rick-and-morty-app

Aplicação Vue.js com Quasar Framework e Apollo Client para consumir a API do Rick and Morty.

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```
